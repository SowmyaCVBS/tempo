Plug Tempo into your private repo

Your tasks live in your private tempo-data repo. Paste a token once and this browser stays synced. Nothing sensitive is stored in the app or its public repo.

Defaults to 30 days out. Tempo warns you a week before it lapses, so syncing never silently stops.

Advanced — change owner / repo / branch
  1. Create a private repo named tempo-data, and upload the board.json I gave you into it.
  2. Make a fine-grained token here: github.com/settings/personal-access-tokens/new
  3. Repository access → Only select repositories → pick tempo-data.
  4. Permissions → Repository → Contents → Read and write. (That's the only one.)
  5. Generate, copy the github_pat_… value, paste it above, hit Connect.

The token is scoped to only that one repo and is revocable in one click. It's stored solely in this browser and sent only to GitHub over HTTPS. On a shared/work machine, use Disconnect when you're done. — Heads up: this needs to run from your GitHub Pages URL; previewing inside a chat window won't reach GitHub, and that's expected.
